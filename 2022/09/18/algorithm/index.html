前缀和与差分

前缀和

S[i] = a[1] + a[2] + … a[i]
a[l] + … + a[r] = S[r] - S[l - 1]//下标从1开始

作用：快速求一段区间的和。

子矩阵的和

S[i, j] = 第i行j列格子左上部分所有元素的和
以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵的和为：
S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1]

一维差分

给区间[l, r]中的每个数加上c：B[l] += c, B[r + 1] -= c

差分矩阵

给以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵中的所有元素加上c：
S[x1, y1] += c, S[x2 + 1, y1] -= c, S[x1, y2 + 1] -= c, S[x2 + 1, y2 + 1] += c

最小生成树

最小生成树有两种算法，分别是Prim和Kruskal

稠密图采用朴素版Prim，稀疏图用Kruskal,堆优化版的Prim用得不多

朴素版prim算法 —— 模板题

时间复杂度是 O(n2+m), n 表示点数，m 表示边数

思路：

  1. 遍历每个点，找到集合外距离最近的点，t。
  2. 用t更新其他点到集合的距离，然后把t加到集合当中。

Kruskal算法 —— 模板题

时间复杂度是 O(mlogm) , n 表示点数，m 表示边数

思路：

  1. 将所有边按照权重大小排序。O(mlogm)
  2. 枚举每条边a,b，权重为c。如果a,b不连通，将这条边也加入到集合当中。O(m)
